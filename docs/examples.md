# Examples

## Visual showcase

The cover figure below is generated by the
[`examples/about-the-data/`](https://github.com/random-walks/nyc-geo-toolkit/tree/main/examples/about-the-data)
example, which exercises every packaged boundary layer and spatial helper in a
single script.

![nyc-geo-toolkit cover figure](assets/images/generated/nyc-geo-toolkit-cover.png)

### Panel breakdown

| Panel | Layer | Features | API demonstrated |
|-------|-------|----------|------------------|
| Boroughs | `borough` | 5 | `load_nyc_boundaries_geodataframe()`, `to_web_mercator()`, `add_osm_basemap()` |
| Community Districts | `community_district` | 71 | `load_nyc_boundaries_geodataframe()` |
| Council Districts | `council_district` | 51 | `load_nyc_boundaries_geodataframe()` |
| NTAs | `neighborhood_tabulation_area` | 195 | `load_nyc_boundaries_geodataframe()` |
| Census Tracts (clipped) | `census_tract` | varies | `load_nyc_boundaries()`, `clip_boundaries_to_bbox()` |
| Geodesy Helpers | n/a | 3 circles | `build_circle_polygon()`, `walk_radius_meters()`, `haversine_distance_meters()` |

### Generating the figure

```bash
cd examples/about-the-data
uv run python main.py
# -> artifacts/nyc-geo-toolkit-cover.png
```

## Self-contained examples

Each example in the `examples/` directory is a standalone project with its own
`pyproject.toml` and can be run independently.

### About the Data

Kitchen-sink visualization of every boundary layer and spatial helper. Generates
a multi-panel cover figure.

```bash
cd examples/about-the-data && uv run python main.py
```

### Boundary Quickstart

Loads a single borough boundary and exports it as GeoJSON.

```bash
cd examples/boundary-quickstart && uv run python main.py
```

### Normalization Demo

Shows how `nyc-geo-toolkit` normalizes inconsistent layer names and geography
values into canonical forms.

```bash
cd examples/normalization-demo && uv run python main.py
```
