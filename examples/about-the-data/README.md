# About the Data

Kitchen-sink example that generates a multi-panel cover figure showcasing every
boundary layer and spatial helper in `nyc-geo-toolkit`.

## Run

```bash
uv run python main.py
```

## Output

- `artifacts/nyc-geo-toolkit-cover.png` -- polished 2x3 panel figure
- `reports/about-the-data-tearsheet.md` -- summary of what was generated
