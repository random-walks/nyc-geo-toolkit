# Examples

`examples/` contains self-contained consumer projects for `nyc-geo-toolkit`.

Start with:

- `examples/boundary-quickstart/`
- `examples/normalization-demo/`
- `examples/boundary-explorer-tearsheet/` — factor-factory + jellycell interop
  showcase (requires the `spatial` extra plus
  [factor-factory](https://github.com/random-walks/factor-factory) and
  [jellycell](https://github.com/random-walks/jellycell))
- `examples/example-template/`
